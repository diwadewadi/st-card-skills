# st-card-skills

Claude Code skills for SillyTavern character card and world book editing.

## Install

```bash
npx st-card-skills
```

This copies skill files to `~/.claude/commands/st/`. Restart Claude Code after installation.

## Available Skills

| Skill | Description |
|-------|-------------|
| `/st:setup` | Install st-card-tools CLI and configure SillyTavern path |
| `/st:help` | Show all available commands and usage |

## Requires

- [st-card-tools](https://www.npmjs.com/package/st-card-tools) — CLI tool (installed via `/st:setup`)
